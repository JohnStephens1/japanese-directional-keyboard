quick overview:

modifiers:
w -> u
d -> e
s -> o
a -> i

q -> soft
e -> hard

special:
xた -> っ
xな -> ん

NumpadPlus -> Backspace (delete last character)


shortcuts:
NumLock on/off -> hiragana/katakana toggle
Ctrl + NumLock -> conversion toggle
Ctrl + Escape -> exit application


full documentation at:
https://github.com/JohnStephens1/japanese-directional-keyboard
